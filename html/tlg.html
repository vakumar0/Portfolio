Description

Think, Learn & Grow (TLG), is an academic project, developed by me while pursuing Post Graduation. It aims to provide an environment where the user can learn at their fullest potential.

TLG at its core is a web browser, developed using Python PyQt libraries. It contains a web-crawler and has the ability to web-scrape. This is achieved by using a third-party library, named, Beautiful Soup. By integrating all these capabilities, TLG provides suggestions to the user based on their searches.Â
