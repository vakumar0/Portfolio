Polaris

Polaris is an IVP product which enables hedge funds to support a host of middle office functions on a single, integrated database platform. It is a real-time warehousing module which offers live ticking Dashboards with P&L Analytics, Performance Calculations and Compliance Alerts.Learn more about the product.
